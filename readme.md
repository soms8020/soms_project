## 🎯 프로젝트 개요

### 서비스명

**“MBTI Lite+ — 나를 피드에 공유하는 5분 테스트”**

### 핵심 목표

* 8~12개의 간단한 질문으로 MBTI 유형을 빠르게 진단
* 결과 페이지에서 **SNS 피드 공유 기능** 제공 (인스타그램/트위터/스레드/카카오톡)
* 공유 시 자동으로 **결과 카드 이미지 생성 (OG 이미지 or Canvas 캡처)**

### 핵심 가치

> “나를 표현하는 테스트” → “나를 공유하는 콘텐츠”
> → **MZ세대의 자기표현 욕구 + SNS 중심 소비 패턴**에 최적화

## 🌐 SNS 공유 기능 상세

### 🔹 공유 방식 1: **Web Share API (모바일 브라우저용)**

```js
navigator.share({
  title: "내 MBTI는 ENFP 💫",
  text: "나의 성향이 궁금해? 5분만에 테스트해봐 👉",
  url: "https://mbti-lite.app/result/enfp"
});
```

### 🔹 공유 방식 2: **html2canvas 캡처 + 다운로드**

* 결과 카드 영역을 `html2canvas`로 이미지화
* 사용자가 직접 인스타그램/스레드 피드에 업로드 가능

### 🔹 공유 방식 3: **SNS SDK 연동 (선택적)**

* 트위터, 카카오톡 공유 API 연동 (메타태그 기반)
* 공유 시 해시태그 자동 포함:

  ```
  #MBTILite #나의성향테스트 #5분MBTI #나를표현하는방법
  ```
